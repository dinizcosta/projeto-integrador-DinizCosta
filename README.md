Projeto Integrador - Desenvolvimento de Sistemas

Sobre o Projeto
Este repositório foi criado para documentar minha jornada no **Projeto Integrador em Sistemas de Informação**.
O foco principal é a construção de um portfólio que conecte competências técnicas à resolução de problemas reais com empatia.

Objetivo
Demonstrar habilidades em desenvolvimento, organização de código e marketing pessoal, garantindo uma vitrine profissional eficiente no GitHub.

Tecnologias e Habilidades
- Desenvolvimento de Sistemas:** Foco em tecnologias recentes e impacto prático.
- Soft Skills:** Comunicação clara, empatia no design de soluções e resiliência diante de desafios técnicos.

Este portfólio é um diferencial competitivo para o mercado de trabalho de TI.
